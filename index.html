🏃‍♂️ 运行

首次运行前请手动修改 openpal3.toml，将《仙剑奇侠传三》游戏目录填入：
（请注意反斜杠需要重复两次）

# PAL3.exe 所在的目录
# The folder where PAL3.exe is
asset_path = "E:\\CubeLibrary\\apps\\1000039"

之后运行 openpal3.exe 即可。如果运行时提示 OpenAL 出错，请下载并安装 OpenAL。

🕹 操作

  • 空格键：对话框下一句
  • F键：互动
  • 方向键：跑
  • 1/2/3/4：存档至第1、2、3、4号存档位
